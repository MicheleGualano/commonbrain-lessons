Go text/template: a leading {{- eats the preceding blank line
#go #templates #codegen #whitespace #golden
Rule. In Go text/template a leading {{- trims all preceding whitespace including newlines (and -}} trims what follows). To keep a separator blank line, put it inside the branch right after {{if}}/{{else}}, never before the {{- if}} — otherwise one template edit churns cosmetic-only diffs across unrelated golden files.

Triggers

  • a blank line disappeared from generated output
  • golden/snapshot diff churns across many unrelated files after one template edit
  • added a {{- if}}/{{- else}} and byte-identical output for other cases broke
  • template dash {{- ate a newline or blank line
