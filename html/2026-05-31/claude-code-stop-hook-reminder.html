Recipe for an end-of-session reminder hook: once-per-session, work-gated, loop-safe
#agent-hooks #automation #process #session-lifecycle
Rule. Event-driven automatic behavior in a coding agent belongs in a hook, not memory. For an end-of-work nudge use a session-stop hook whose script: (1) exits 0 at once if stop_hook_active is true, breaking the block-then-re-stop loop; (2) fires once per session via a marker file keyed on the session id; (3) proceeds only if the transcript shows a file-edit tool ran, a proxy for the unmeasurable I-learned-something; (4) returns a block decision with a reason so the nudge re-enters the model's context. Never block the flow: on any error, exit 0. Validate by piping it the stdin JSON it receives.

Triggers

  • want an automatic end-of-session reminder or action in a coding agent
  • stop hook fires every turn and is too noisy
  • stop hook infinite loop
  • stop_hook_active
  • automate a recurring end-of-work behavior that needs a hook, not memory
