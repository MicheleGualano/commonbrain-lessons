errorlint: wrap errors with %w, not %v (multi-%w ok since Go 1.20)
#go #errors #errorlint #linter
Rule. Wrap errors with %w (not %v) in fmt.Errorf so errors.Is and errors.As can unwrap them; %v formats the text but breaks the chain. Multiple %w in a single fmt.Errorf is allowed since Go 1.20.

Triggers

  • non-wrapping format verb for fmt.Errorf. Use %w to format errors (errorlint)
  • errors.Is or errors.As never matches a wrapped error
  • an error formatted with %v inside fmt.Errorf
